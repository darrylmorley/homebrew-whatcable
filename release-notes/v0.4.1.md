## What's fixed

- **M1 / M2 USB-C port detection** ([#2](https://github.com/darrylmorley/whatcable/issues/2)). Earlier builds showed "No USB-C ports detected" on every M1 and M2 generation. The IOKit class for a physical port node is `AppleTCControllerType10` on M1/M2 and `AppleHPMInterfaceType10/11/12` on M3-era — WhatCable now matches both.

## Install

Download `WhatCable.zip`, unzip, and drag `WhatCable.app` into `/Applications`. Notarised + stapled — no Gatekeeper warning.

If you have an earlier version installed, it will offer to update itself in-app the next time it polls (or via the menu).
