## What's new

- **Quit button in the main view.** A small Quit button now sits in the bottom-left corner of the popover, so you can exit the app without needing to find the right-click menu on the status bar icon.

## What's fixed

- **Dock mode menus now match menu bar mode.** Check for Updates, About WhatCable, and the GitHub link were missing from the app menus when running in Dock mode. All three are now present and consistent.
- **In-app updater security hardened.** The updater now requires the exact asset name, validates the download URL, checks the zip for unsafe paths before extracting, verifies the bundle ID and Gatekeeper acceptance before installing, and only strips quarantine after all checks pass. Thanks to @durul for the detailed audit and @0x687931 for independently implementing the same fixes.

## Install

`brew upgrade --cask whatcable`, or download `WhatCable.zip` from this release page and drag the new `.app` into `/Applications`.
