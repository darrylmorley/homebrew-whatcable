## What's new

- **⌥-click for technical details.** Hold Option while clicking the menu bar icon to open the popover with all the advanced fields visible — matches the convention used by macOS's own Wi-Fi / Volume / Bluetooth menus. Plain click is unchanged. The Settings toggle remains as a discoverable backup. Idea ported from [#4](https://github.com/darrylmorley/whatcable/pull/4) by @rolandgroen.
- **GitHub link** in the right-click menu — "WhatCable on GitHub" opens the repo. From [#7](https://github.com/darrylmorley/whatcable/pull/7) by @rolandgroen.

## Install

Download `WhatCable.zip`, unzip, and drag `WhatCable.app` into `/Applications`. Notarised + stapled — no Gatekeeper warning.

If you have v0.4.0 or later installed, the in-app updater will offer to install this version automatically.
