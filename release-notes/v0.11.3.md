# More accurate cable and port detection

## What's fixed

- **Unbranded cables are no longer mislabeled as an Anker 140W.** Cables that report an all-zero fingerprint (no real e-marker data, common on cheaper cables) were all collapsing onto one curated cable entry and being shown as that product. They now correctly show as unidentified. Surfaced by community cable reports.

- **Cable identity and power data now line up to the correct port.** On some Macs the e-marker details and the live power data for a port could resolve to different ports, so a port could show mismatched information. All port data now correlates to the correct physical port.
