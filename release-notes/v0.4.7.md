## What's fixed

- **Wrong empty state with "Hide empty ports" on.** When nothing was plugged in, the popover showed "No USB-C ports detected" along with a diagnostic about missing port-controller services. That message was misleading: the ports were detected, they were just being hidden by the filter. The empty popover now distinguishes two cases:
  - "No USB-C ports detected" appears only when the IOKit query genuinely returns nothing.
  - "Nothing connected" appears when ports exist but the Hide empty ports toggle is filtering them all out, with a pointer at the toggle.

## Install

Download `WhatCable.zip`, unzip, and drag `WhatCable.app` into `/Applications`. Notarised and stapled, no Gatekeeper warning.

If you have v0.4.0 or later installed, the in-app updater will offer to install this version automatically.
