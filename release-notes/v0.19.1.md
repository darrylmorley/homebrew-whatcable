# No more cables shown under the wrong brand

A cable that doesn't report a vendor ID could be matched to the wrong maker (for example a Statik cable showing as "Anker"). That happened because the match leaned on the cable's capability data, which many unrelated cables share.

## What's fixed

- **Cables are now identified only by their vendor and product ID.** A cable can no longer be labelled as a brand it doesn't carry. One that reports no vendor ID simply shows what it can do (speed, power) without guessing a brand.

## Install

`brew upgrade --cask whatcable`, download from [Releases](https://github.com/darrylmorley/whatcable/releases), or update in the app.
