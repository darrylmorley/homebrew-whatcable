# Tap any device to see inside it

The Diagnostics screen now expands each connected device to show its full detail, no more guessing what a row actually is.

## What's new

- **Tap a device on the Diagnostics screen to open it up.** Every USB device in the list (hubs, adapters, dock chips, Billboard devices) now expands to show its vendor and product ID, USB version, device class, the power it asks for versus what the port offers, serial number, and where it sits on the bus. There's a raw-properties view tucked behind its own toggle for the deep technical detail, and a Billboard device gets a plain note explaining what it is. The data was always being read; now you can actually see it.

## Install

`brew upgrade --cask whatcable`, download from [Releases](https://github.com/darrylmorley/whatcable/releases), or update in the app.
