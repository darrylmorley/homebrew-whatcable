## What's new

- **Settings panel** — toggles moved out of the footer into a dedicated panel, reached via a new gear button in the top-right. Grouped into Display / Behavior / Notifications, with room to grow.
- **Hide empty ports** ([#6](https://github.com/darrylmorley/whatcable/issues/6)) — new toggle in Settings → Display. When enabled, the popover only shows ports that currently have something connected.

## Install

Download `WhatCable.zip`, unzip, and drag `WhatCable.app` into `/Applications`. Notarised + stapled — no Gatekeeper warning.

If you have v0.4.0 or v0.4.1 installed, the in-app updater will offer to install this version automatically.
