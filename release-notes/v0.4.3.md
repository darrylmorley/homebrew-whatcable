## What's new

- **Optional window mode** ([#1](https://github.com/darrylmorley/whatcable/issues/1)) — new "Show in menu bar" toggle in Settings → Behavior. Turn it off and WhatCable runs as a regular Dock app with a window instead of a permanent menu bar icon. Switches live, no relaunch.

## Install

Download `WhatCable.zip`, unzip, and drag `WhatCable.app` into `/Applications`. Notarised + stapled — no Gatekeeper warning.

If you have v0.4.0 or later installed, the in-app updater will offer to install this version automatically.
