# Sharper widgets and a steadier Power Monitor

This release sharpens the desktop widgets, tidies the Power Monitor, and folds in three rounds of community translation work.

**Widgets**
- Each port now shows a colour-coded link-speed badge (480M / 5G / 10G / 20G / 40G / 80G), so you can see at a glance how fast a cable actually linked up.
- The "Display connected" card now shows resolution and refresh rate, plus the monitor's name where macOS exposes it (for example "Studio Display · 5K 60Hz").
- A cleaner, unified look across all widget sizes: consistent pills, accent bars, and tighter spacing.

**Power Monitor widget**
- Fixed the live power graph flickering on and off. WidgetKit can't refresh fast enough for a true live chart, so the widget now shows steady wattage numbers instead (charger, system draw, and per-port watts). Live, second-by-second tracking stays in the in-app Power Monitor window.

**Translations**
- Korean: a consistency and quality pass across the whole set, thanks to @dohun0310.
- Simplified Chinese: filled in the remaining English placeholders and added plural rules, thanks to @carlchina.
- Italian: refinements to charger, current, and cable-detail wording.
