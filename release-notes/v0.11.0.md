# Power input monitoring and accurate per-port power

## What's new

- **Live system power input.** The Power Monitor now charts the power coming into your Mac (voltage, current, watts) over time, so you can see real draw even on Macs that do not expose live per-port metering.

- **Every port, correctly identified.** The Power Monitor now lists every physical port with its correct name and its real negotiated power contract. A charging USB-C port that previously could go missing, or show under the wrong name (for example "MagSafe"), now appears correctly with its true voltage, current and wattage.

- **Diagnostic data collection.** A new opt-in test kit can gather and submit anonymised port and power diagnostics, making it easier to improve hardware coverage from real-world reports.

## What's fixed

- **Negotiated PD contract is read, not guessed.** On Macs without live per-port metering, the per-port card now decodes the real negotiated voltage and current from the port's own power data instead of leaving it blank, clearly labelled as the agreed contract rather than a live measurement.

## Install

`brew upgrade --cask whatcable`, or download `WhatCable.zip` from this release page and drag the new `.app` into `/Applications`.
