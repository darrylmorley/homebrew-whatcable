## What's fixed

- **MagSafe port now appears on M2 MacBook Air.** The watcher was matching `AppleTCControllerType10` (M1/M2 USB-C) but not its sibling `AppleTCControllerType11`, which is the class M2 MBA uses to register its MagSafe 3 port. Adding it to the match list makes MagSafe show up in the popover and CLI on M2 MBA exactly as it already does on M3 and later. Thanks to @bigskookum for the original report and to Michael for the ioreg dump that pinned down the class (#13).

No other changes since v0.5.4.

## Install

`brew upgrade --cask whatcable`, or download `WhatCable.zip` from this release page and drag the new `.app` into `/Applications`.
