# Desktop widgets

## What's new

- **Desktop widgets.** Three sizes: small (single port), medium (all ports at a glance), and large (all ports with full detail). The widgets update live whenever cable state changes. Add them from the macOS widget gallery under "WhatCable".

- **Device count badge.** Ports with USB devices connected show a small count indicator on all three widget sizes.

- **Confirmed TB5 labels.** Thunderbolt 5 links now show "Thunderbolt 5" instead of the hedged "Unknown generation (inferred TB5)" wording. Thanks to contributors who shared TB5 endpoint data on #52.

- **Cable database page.** Known cables reported via the in-app flow are now listed at [whatcable.uk/cables](https://whatcable.uk/cables), searchable by brand or vendor ID.

## Other

- Docs and CLI documentation pages added to whatcable.uk.
- Footer and about panel now show "WhatCable" instead of a personal name.
- Cable report sync script for maintaining the known-cables database.
