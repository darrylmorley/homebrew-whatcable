# Your 5K display reads as 5K

WhatCable now reads your display's live mode straight from macOS. 5K and 6K screens show their real resolution instead of 4K, and a display that's already at its best gets a confident "running at full quality" instead of a maybe. A cable-trust false alarm is fixed too.

## What's new

- **5K and 6K displays show their true resolution.** WhatCable used to work out a display's top mode from its EDID (the spec sheet it sends down the cable), but that format can't describe a 5K mode, so Apple's 5K and 6K screens came up as 4K. It now reads the live mode straight from macOS, so a Studio Display reads 5120 × 2880 (#249).
- **A display at its best mode is now confirmed, not hedged.** When a monitor reaches its top mode using compression (DSC), the link rate on its own can't prove it's at full quality. WhatCable now reads the live mode and says "running at full quality" outright, instead of "may be using compression". Builds on the quieter display check from the last release (#246).

## What's fixed

- **A cable with a blank maker ID is no longer flagged as a possible counterfeit.** Some genuine cables leave that field empty. WhatCable was reading the blank as a mismatch and raising a trust flag it shouldn't have (#228).
