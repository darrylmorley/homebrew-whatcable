## What's fixed

- **Cmd+, now opens the Settings panel correctly.** Previously pressing Cmd+, opened a blank empty window. It now opens the same in-place settings panel as the gear icon. The right-click status menu also has a new "Settings..." item. Thanks to @blech.

## Install

`brew upgrade --cask whatcable`, or download `WhatCable.zip` from this release page and drag the new `.app` into `/Applications`.
