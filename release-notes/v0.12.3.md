# Refined Italian translation

## What's changed

- **Italian translation refresh.** Thanks to @bovirus for a thorough pass: `caricabatterie` is now `caricatore` throughout (more accurate for general USB-C PD chargers), and the Negotiation Diagnostics and Pro overview screens are now fully translated rather than showing English placeholders.

- **Dropped the PRO badge from the popover footer.** Bit much for day-to-day use. The Pro overview screen is still one tap away from any locked feature.

## Install

- **Inside the app:** right-click the menu bar icon and choose "Check for Updates…". WhatCable also checks for updates on its own and posts a system notification when a new version is out.
- **Homebrew:** `brew upgrade --cask whatcable`.
- **Manual:** download `WhatCable.zip` from this release page and drag the new `.app` into `/Applications`.
