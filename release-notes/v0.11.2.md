# Every screen now translated in all languages

## What's fixed

- **No more English fallback.** The Diagnostics and Power Monitor windows, the cable-speed and connected-device lines, and the "Contribute Diagnostic Data" privacy screen still showed English in some languages. Every screen is now fully translated across all eleven languages: English, Italian, Polish, Armenian, Simplified Chinese, German, Spanish, French, Hindi, Japanese and Norwegian Bokmål. Community refinements are welcome via the public repo.

## Install

`brew upgrade --cask whatcable`, or download `WhatCable.zip` from this release page and drag the new `.app` into `/Applications`.
