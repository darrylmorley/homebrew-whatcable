# Footer toggle restored, resilient updater

## What's new

- **Technical details toggle back in the footer.** The "Show technical details" switch used to live in the footer for quick access, but got removed when Settings was introduced. It's back. The toggle still lives in Settings too, so both paths work. Quit button moved to the right-hand side of the footer.

- **Updater bundle replacement is now recoverable.** If the self-updater fails mid-replace (crash, permission error, disk full), the original app bundle is restored from a backup copy instead of leaving a broken install. Covers the window between removing the old bundle and moving the new one into place.

## Other

- Landing page: added meta tags, schema markup, and new content sections.
