# A clearer path to Pro, plus charger brand info

## What's new

- **A real Pro overview screen.** Tapping a Pro feature without an active licence used to show a one-line "Pro required" message and nothing else. There is now a proper in-app screen that explains what Pro includes, shows the licence key field, and links straight to the buy page. The same screen is reachable any time from a small "PRO" pill in the popover footer, which hides itself once you activate. No more hunting through Settings to figure out how to upgrade.

- **Charger brand and product name.** Free tier. When a recognised charger is plugged in, the port summary now shows the brand and product name, not just the wattage. Apple chargers, third-party USB-C chargers with proper PD identity, and Mac-internal adapters are all covered. Falls back gracefully when the charger does not advertise enough to identify it.

- **Traditional Chinese localisation.** WhatCable is now available in 繁體中文 (zh-Hant) alongside the existing 11 languages.

## What's fixed

- **Pro screen headers no longer float mid-window.** The Power Monitor and Negotiation Diagnostics screens were vertically centring their headers when there were no connections, so the page title sat roughly a quarter of the way down the popover. Headers now pin to the top where they belong.

- **Cable database: a few curated entries were colliding.** Some cables sharing a vendor / product ID were overwriting each other's notes in the bundled database. The sync now keeps each curated row distinct.

## Housekeeping

- The tagline subtitle was removed from the popover header so it stays compact and leaves more room for the Pro feature buttons.

## Install

- **Inside the app:** right-click the menu bar icon and choose "Check for Updates…". WhatCable also checks for updates on its own and posts a system notification when a new version is out.
- **Homebrew:** `brew upgrade --cask whatcable`.
- **Manual:** download `WhatCable.zip` from this release page and drag the new `.app` into `/Applications`.
