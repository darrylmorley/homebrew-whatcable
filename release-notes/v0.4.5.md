## What's fixed

- **"Charging well" reported on disconnected ports** ([#9](https://github.com/darrylmorley/whatcable/issues/9)). MagSafe (and some USB-C) port controllers cache the last negotiated PDO and keep advertising it long after the charger is unplugged, so the diagnostic was reading stale 94W "winning" data on an empty port. WhatCable now gates the charging diagnostic on the port-level ConnectionActive flag, which correctly reports "No" when nothing is connected. Verified on M1, M2, and M5.

## Install

Download `WhatCable.zip`, unzip, and drag `WhatCable.app` into `/Applications`. Notarised and stapled, no Gatekeeper warning.

If you have v0.4.0 or later installed, the in-app updater will offer to install this version automatically.
