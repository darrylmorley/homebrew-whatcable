# Bundle ID changed

WhatCable now uses the bundle ID `uk.whatcable.whatcable`, matching the new `whatcable.uk` domain.

This is a one-time identity change before App Store submission. Existing installations will reset preferences and notification permissions on first launch of this version. If you had launch at login enabled, re-enable it from Settings after upgrading.

Users upgrading through Homebrew or by replacing `WhatCable.app` manually should not need to do anything else. The old bundle ID was `com.bitmoor.whatcable`.

If the in-app updater from an older release rejects this update with an unexpected bundle identifier message, install this version through Homebrew or from the GitHub release download instead. This should only affect the bundle ID transition release.
