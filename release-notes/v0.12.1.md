# Smaller, sharper app icon

## What's fixed

- **App icon no longer oversized.** The blue squircle now sits inside the standard macOS safe-area margin, so WhatCable matches the visual size of other dock and Launchpad icons (issue #164). The cable symbol inside the squircle is slightly smaller too, sitting comfortably in the safe area instead of crowding the corners.

## Housekeeping

- The WhatCable Pro source now ships under its own proprietary licence, separate from the MIT licence covering the free / open-source code. No change for end users; the runtime Pro licence-key terms published at [whatcable.uk](https://whatcable.uk) are unchanged.

## Install

`brew upgrade --cask whatcable`, or download `WhatCable.zip` from this release page and drag the new `.app` into `/Applications`.
