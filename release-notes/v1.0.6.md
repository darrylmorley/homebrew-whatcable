# Devices behind a Thunderbolt dock or display now show up

If you run a Thunderbolt dock or a display like the Studio Display, the USB things plugged into it (your mouse, keyboard, and so on) now appear in WhatCable. Before, they were counted in the total at the bottom but listed nowhere. Plus Italian translations.

## New

- **USB devices behind a Thunderbolt dock or display are now listed.** Plug a mouse and keyboard into a Studio Display, for example, and they show up: nested under that display's port when it's the only Thunderbolt device connected, or in an "Other USB devices" section when you have more than one dock or display attached. These come in over Thunderbolt rather than a plain USB-C port, so there is no cable or power data for them, just the device and its speed. Thanks to @robertovalerio for the request and for the data it was built on (M-series MacBook plus a Studio Display).
